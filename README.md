# GeoVisual Stories — Geospatial Portfolio Website

A personal portfolio website showcasing work in GIS, cartography, spatial analysis, and data visualization.

🌐 Live site: https://geovisualstories.github.io/

---

## Overview

This website highlights geospatial projects focused on:

- GIS and cartographic design  
- Spatial analysis and modeling  
- Environmental and energy applications  
- Data visualization and storytelling  

The goal is to present technical work in a way that is both rigorous and visually clear.

---

## Featured Work

- **Geothermal Suitability Mapping**  
  Spatial modeling for geothermal resource potential and uncertainty.

- **Groundwater & Hydrologic Mapping**  
  GIS workflows for groundwater elevation and hydrologic interpretation.

- **Environmental Data Visualization**  
  Python-based visualizations for communicating complex datasets.

## Tech Stack

- HTML5  
- CSS3 (custom styling, responsive design)  
- JavaScript (interactivity, animations)
